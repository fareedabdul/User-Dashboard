# 🚀 User Directory Dashboard

A responsive React application that displays users with search, sorting, and detailed views.

## 🔥 Features
- User Dashboard (Table + Mobile Cards)
- Search by Name & Email
- Sorting (Name & Company)
- User Detail Page
- Fully Responsive UI

## 🛠 Tech Stack
- React.js
- Tailwind CSS
- React Router
